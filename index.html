Meet Luna! üéí She's a curious girl who just got the most amazing gift ‚Äî a magic backpack! This isn't any ordinary backpack. It has special pouches that can hold anything, and each pouch has a name tag.

Luna enters a cozy kitchen and meets Chef Maya! üë©‚Äçüç≥ Maya has a magical recipe book ‚Äî each recipe is a set of steps she can use anytime. In Python, a recipe is called a function!

You create a function with def (short for "define"). Give it a name, add ingredients (called parameters), and write the steps inside!

Now Maya can use this recipe anytime by calling it! She just says the recipe name and gives it an ingredient:

Functions are super powerful! Write the recipe once, use it a million times. That's the magic of code reuse! ‚ú®

In a cozy bedroom, Luna meets Zara and her magical toy box! üß∏ This isn't just any box ‚Äî it has numbered slots where each toy has its own spot. In Python, this is called a list!

A list holds items in order, and each item has a number (starting from 0!). You can add toys, remove them, or grab any toy by its number!

# Zara's toy box (a Python list!) toys = ['teddy', 'robot', 'dragon', 'puzzle'] print(toys[0]) # teddy (first!) print(toys[2]) # dragon

Zara can add new toys with append(), remove toys with remove(), and check how many she has with len()!

# Add a unicorn toy! toys.append('unicorn') print(len(toys)) # 5 toys now! # Remove the robot toys.remove('robot') print(toys) # ['teddy', 'dragon', 'puzzle', 'unicorn']

You can even grab a slice of toys! toys[1:3] gives you toys at positions 1 and 2. Lists are one of the most useful things in Python! üéÅ
